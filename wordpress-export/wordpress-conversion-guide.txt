CASA DA PEIXE & LAPICANHA - WORDPRESS CONVERSION GUIDE
=======================================================

This guide will help you convert your React restaurant website to WordPress.

STEP 1: WORDPRESS SETUP
------------------------
1. Purchase WordPress hosting (recommended: SiteGround, Bluehost, WP Engine)
2. Install WordPress on your domain
3. Choose a restaurant theme or create custom theme
4. Install essential plugins:
   - Contact Form 7 (for reservations)
   - WP Restaurant Menu (for menu display)
   - Yoast SEO (for search optimization)
   - WooCommerce (if you want online ordering)

STEP 2: IMPORT YOUR DATA
------------------------
1. Use the data from 'restaurant-data.php'
2. Create WordPress pages:
   - Home page
   - Casa Da Peixe page
   - Lapicanha page
   - Contact page
   - Menu pages

3. Import menu items:
   - Use a restaurant menu plugin
   - Or create custom posts for each menu item
   - Copy prices and descriptions exactly

STEP 3: STYLING
---------------
1. Copy CSS from 'theme-colors.css' into your WordPress theme
2. Add Portuguese restaurant colors and fonts
3. Create responsive design for mobile
4. Test on all devices

STEP 4: FUNCTIONALITY
---------------------
RESERVATIONS:
- Install Contact Form 7
- Create reservation forms for each restaurant
- Set up email notifications
- Test form submissions
